{% markdown %} # {{page.title}} __TL;DR;__ `npm install -g quickshot` {% endmarkdown %}
{% markdown %} ## Verbose - Quickshot is written in [Iced CoffeeScript](http://maxtaco.github.io/coffee-script/) and runs on [Node.js](https://nodejs.org/) and [io.js](https://iojs.org/). - If you have neither of those installed visit [https://nodejs.org/download/](https://nodejs.org/download/) - After installing one of the above, open a terminal and run `npm install -g quickshot` - If you have errors, you may need to run the above command with `sudo`. Try `sudo npm install -g quickshot` [Continue to "Getting Started"](/getting-started.html) {% endmarkdown %}
