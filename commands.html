{% markdown %} # {{page.title}} Executing `quickshot help` at any time will provide an overview of the available commands: ``` quickshot configure Creates/Updates the configuration file in current directory quickshot download [filter] Download theme files, optionally providing a filter quickshot upload [filter] Upload theme files, optionally providing a filter quickshot watch Watch project folder and synchronize changes automatically quickshot --help Show this screen. ``` {% endmarkdown %}
